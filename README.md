# Phishing Simulator: Cyber Awareness Training Tool

A Python-based MVP for simulating phishing scenarios to train users on spotting scams. Generates fake emails/websites, quizzes, and scores responses.

## Features (MVP)
- Random phishing examples (emails, sites).
- Interactive quiz.
- Scoring and tips.

## Setup
- Clone and run `python src/main.py`.
